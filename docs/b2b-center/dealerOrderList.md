## 1.提供企业购中台-查看经销商EUC商机列表

### 1.1 地址
> baseUrl + /api/eucapp/euc_list

### 1.2 请求方式
> POST

### 1.3 参数

header

|  参数   | 注释  |是否必填  |备注  |
|  ----  | ----  |----  |----  |
| Authorization  | tooken | 是 |身份验证|

param

|  参数   | 注释  |是否必填  |备注  |
|  ----  | ----  |----  |----  |
| businessName  | 商机名称 | 否 |-|
| companyNameA  | 甲方公司名 | 否 |-|
| businessCode  | 商机编码 | 否 |-|
| ifTopCustomer  | 是否TOP | 否 |1是;0否|
| topCustomerAbbreviation  | TOP名称 | 否 |-|
| industryId  | 行业id | 否 |-|
| startEntryDate  | 开始时间 | 否 |-|
| endEntryDate  | 结束时间 | 否 |-|
| orderType  | 类型 | 否 |1抢单;0指派|
| businessType  | 商机类型 | 否 |新建在建项目；精装修项目；土地成交项目|
| undertake  | 承接方式 | 否 |1零售;0工程单;99无|
