## 6.提供大客户app-euc抢单列表

### 6.1 地址
>  baseUrl + /api/eucapp/iteration/garbList

### 6.2 请求方式
> POST

### 6.3 参数

|  参数   | 注释  |是否必填  |备注  |
|  ----  | ----  |----  |----  |
| orgId  | 中心编码 | 是 |- |
| gridCode  | 网格编码 | 是 |- |
